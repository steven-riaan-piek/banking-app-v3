Banking App V3 (Still a C# Console App)

Overview
I expanded my banking application into a multi-user system that allows multiple accounts to be created, managed, and interacted with. This version introduces account management and fund transfers between users.

---

Features
- Multi-user account system
- User login (basic)
- Deposit and withdraw funds
- Transfer money between accounts
- Transaction history tracking
- Data persistence using file storage
- Input validation and error handling

---

Technologies Used
- C#
- .NET Console Application
- File handling (System.IO)
- Collections (Dictionary for managing users)

---

What I Learned
- Managing multiple users using data structures (Dictionary)
- Designing systems with interacting components
- Implementing fund transfers between accounts
- Improving program structure and scalability
- Thinking in terms of real-world system behavior
